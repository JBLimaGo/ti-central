# TI Central — Gestão de Chamados

Sistema web de gestão de chamados de TI para empresa de maquiagem com 9 lojas + CD.

## Acesso
🔗 **[Abrir Sistema](https://jblimago.github.io/ti-central)**

## Funcionalidades
- 📊 Dashboard com KPIs em tempo real
- 🎫 Registro e acompanhamento de chamados
- 💬 Histórico de cobranças ao suporte (WhatsApp)
- 🖼 Anexo de prints de conversa por cobrança
- 🏪 Análise por loja
- 📋 Relatório para supervisor
- 📱 Responsivo — funciona no celular e computador

## Tecnologia
- Frontend: HTML/CSS/JS puro (sem dependências)
- Backend: Supabase (PostgreSQL na nuvem — São Paulo)
- Hospedagem: GitHub Pages
